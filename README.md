## 写作习惯

- 资讯类笔记优先简报化：先结论，再动态，再趋势判断。
- 工具类笔记优先保留个人判断，不写成说明书堆砌。
- 设计类笔记尽量沉淀成可复用工作流，而不是只记录概念。
