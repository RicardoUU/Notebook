## 重点动态

### 1. GitHub Copilot CLI 支持 BYOK 和本地模型
- GitHub 在 **2026-04-07** 发布更新，Copilot CLI 现在支持自带 key，并可接入本地模型。
- 这意味着团队更容易把 CLI 工作流接进自有网关、代理层或本地推理环境，而不是只能走 GitHub 托管模型。
- 信号评级：高
- 原始链接：https://github.blog/changelog/2026-04-07-copilot-cli-now-supports-byok-and-local-models

### 2. Anthropic 推出 Project Glasswing
- Anthropic 在 **2026-04-07** 公布 `Project Glasswing`，重点是把模型能力用于检测和缓解网络攻击、欺诈与滥用。
- 这说明 AI 厂商开始把“安全能力”当成前台产品能力，而不只是合规附属项。
- 信号评级：中
- 原始链接：https://www.anthropic.com/news/project-glasswing
