## 我现在的使用判断

- Claude Code 更适合处理长链路工程任务，而不是只当聊天工具用。
- `CLAUDE.md`、skills、plugins、subagents 这些东西，本质都是在给 Claude 补“工作环境”和“工作纪律”。
- 复杂任务里，最重要的不是让 Claude 立刻写代码，而是让它先理解、规划、验证、review。
